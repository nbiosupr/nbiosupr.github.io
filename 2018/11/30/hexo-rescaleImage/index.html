Hexo, Asset 폴더애 있는 이미지를 크기 조절하여 포스팅하기

Posted on 2018-11-30 | In hexo

Asset 자원 폴더에 있는 이미지의 크기를 변경하여 포스트에 불러올 수 있는 방법에 대해 알아봅시다.

들어가며

hexo는 각 포스트 별로 자원을 관리할 수 있도록 각각의 폴더를 제공합니다. 이 자원들을 어떻게 사용할 수 있을까요?

hexo의 Official 페이지에서는 다음과 같은 태그 플러그인을 제공하고 있습니다.

이 태그들을 사용하여 편리하게 자원들을 불러올 수 있지만, 이 태그들은 아쉬운 점을 가지고 있습니다.

이미지 자원을 불러오는 는 아쉽게도 이미지 스케일 옵션을 가지고 있지 않습니다.

따라서 이미지의 크기를 조정 하면서 포스트에 불러오기 원한다면 다른 방법을 찾아보아야 합니다.

해결 방법

정적 문서로 변환된 후의 자원의 위치

포스트에 필요한 이미지를 각 asset 폴더에 업로드한 후 hexo g 명령어를 실행하여 봅시다.

각 포스트의 asset 폴더는
{hexo 설치폴더}/source/_post/{post title}
에 위치하고 있습니다.

{hexo 설치 폴더}/public/연도/월/일/{post_title} 가 새롭게 만들어 진 것을 확인 하실 수 있습니다.

ex) {hexo_folder}/2018/11/30/newpost

이를 통해 우리는 각 포스트의 asset 폴더에 업로드 되었던 자원들의 정적 주소를 알 수 있습니다.

폴더 안을 들여다 보면 이전에 업로드 했던 이미지를 볼 수 있습니다.

이미지 포스팅

이미지 자원의 정적 주소를 활용하여, 다음과 같은 두 가지 방법으로 이 문제를 해결할 수 있습니다.

  • Markdown 기본 이미지 태그
  • HTML 태그

1. 기본 이미지 마크다운 활용

다들 아시다시피 기본적으로 제공되는 이미지 마크다운은
![description](img_path) 입니다.
여기에 다음과 CSS를 추가하여 너비 속성을 주면 이미지 크기를 조절할 수 있습니다.

width와 height 둘 중 하나의 속성만 줘도 나머지 하나는 자동으로 조절됩니다.

이제 여기에 이전에 확인했던 정적 주소를 img_path로 사용하면 이미지가 원하는데로 포스팅된 것을 확인할 수 있습니다.

2. HTML 태그 활용

이미지를 위한 HTML 태그는
<img src="img_path" heigth="100" width="100> 입니다.

마크다운과 마찬가지로 width와 height 둘 중 하나의 속성만 줘도 나머지 하나는 자동으로 조절됩니다.

마치며

지금까지 제가 검색으로 찾은 내용들을 정리해 보았습니다. 많은 분들에게 도움이 되길 바라겠습니다. 부족한 글 읽어주셔서 감사합니다.
